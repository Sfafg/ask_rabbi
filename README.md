## Prerequisites

You need a working internet connection and the Docker Compose tool.  

On Arch Linux, it can be installed using:

```bash
sudo pacman -S docker-compose
```

## Installation
```bash
git clone https://github.com/Sfafg/ask_rabbi.git
cd ask_rabbi
```

## Running the application

```bash
docker-compose up -d
```
the website should be working at
http://localhost:3000/
if something is not working wait and refresh.
>It is working on my machine, so it should be working on yours as well.
